## Performance Smoke Test Baselines

These are integration tests that exercise the full proxy stack.

### TCP Relay Smoke

- SOCKS5 listener ‚Üí direct TCP echo: ~5000 connections/sec (local)
- Successful connections with payload roundtrip
- No error rate under normal conditions

### UDP Relay Smoke

- Standalone UDP direct relay: ~10000 datagrams/sec (local)
- Flow table entries cleaned up after control connection close
- No silent packet loss under normal conditions

### Resource Leak Baselines

- File descriptors return to pre-test count (¬±2 tolerance for cleanup timing)
- Task tracker drains to zero after session completion
- Active connection count returns to zero

## Caveats

- All baselines are debug profile. Release profile will be 2-5x faster
  for CPU-bound operations.
- Local loopback numbers are not representative of network performance.
- Criterion benchmarks measure micro-op latency, not end-to-end proxy throughput.
- Python binding overhead is measured separately in `PYTHON_BINDING_OVERHEAD.md`.
- Future baselines should record exact Rust compiler version for reproducibility.
